ブログ記事のパフォーマンス・クリック・スクロールをヒートマップで可視化するプラグイン。

== Description ==

HINAKIRA Heatmap は、ブログ記事のユーザー行動を3つの視点で可視化します：

1. **記事パフォーマンスマップ** — どの記事がよく読まれているかをツリーマップで一目で把握
2. **クリックヒートマップ** — ページ上のどこがクリックされているかを可視化
3. **スクロールヒートマップ** — 記事のどこまで読まれているか（読了率）を分析

= 特徴 =

* 軽量なトラッキングスクリプト（約3KB）
* 外部サービス不要 — すべてのデータはWordPress内に保存
* プライバシー配慮 — IPアドレスやCookieは一切保存しない
* 古いデータの自動クリーンアップ
* モダンで使いやすいダッシュボード

== Installation ==

1. プラグインをアップロードして有効化
2. 管理画面の「HINAKIRA Heatmap」メニューからダッシュボードを確認
3. データが溜まるのを待って分析を開始
